# ⚓ 一致性锚 (Consistency Anchor)

[![SillyTavern](https://img.shields.io/badge/SillyTavern-1.13%2B-8A2BE2?style=flat-square&logo=github)](https://github.com/SillyTavern/SillyTavern)
[![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)](./LICENSE)
[![Version](https://img.shields.io/badge/version-1.0.0-blue?style=flat-square)](./manifest.json)

**一致性锚** 是一个为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 设计的 AI 角色扮演质量保障插件。它能有效防止角色在对话中出现**人称漂移、机械复读、格式错乱、设定遗忘、思维链泄露**等问题，让你的角色始终保持一致、沉浸的互动体验。（其实是在作者被大型世界书、角色卡和聊了很多层楼的超长上下文

> 💡 **一句话概括**：给 AI 角色装上了"性格稳定器"和"记忆锚点"。

---

## ✨ 功能特性

| 功能 | 说明 |
| :--- | :--- |
| 🧭 **人称一致性** | 自动检测并提醒角色人称（第一/二/三人称）是否漂移 |
| 📝 **格式自动修正** | 智能统一对话、动作、内心独白的格式（如引号、星号） |
| 🔁 **复读检测** | 识别并警告 AI 机械重复内容，避免循环对话 |
| 🧠 **思维链拦截** | 检测并拦截 AI 泄露内部推理过程 |
| 🗂️ **记忆池注入** | 自动将高优先级世界书内容注入上下文，防止遗忘重要设定 |
| ⚓ **设定锚点注入** | 定期将角色规范注入对话，巩固角色一致性 |
| 🎯 **双入口访问** | 支持底部"魔法棒"菜单 + 可拖拽悬浮按钮，随时调出设置 |
| 💾 **设置持久化** | 所有配置自动保存，下次打开无需重新设置 |
| 📊 **世界书状态栏强制输出** | 开启后，插件会强制要求 AI 按照世界书中定义的格式输出状态栏 |

## 📦 安装

### 方式一：通过扩展面板安装（推荐）

1. 打开 SillyTavern，点击顶部菜单栏的 **扩展** (Extensions) 图标（🧩）。
2. 在扩展管理面板中，点击 **"安装扩展"** (Install Extension)。
3. 输入以下 GitHub 仓库地址：
1.18及以上:https://github.com/px06427/Ganlu-wine
1.13及以上:https://github.com/px06427/st_consistency_anchor
4. 点击 **"安装"**，等待安装完成。
5. 刷新页面即可使用。

### 方式二：手动安装（适合开发者或离线环境）
1. 进入 SillyTavern 的 `data/<你的用户名或者是default-user>/extensions/` ，在extensions里创建一个新文件夹：consistency_anchor。
2. 将本插件克隆或下载到该文件夹(consistency-anchor)里。

1. 最终路径应为：
   ```
   data/<你的用户名或者default-user>/extensions/consistency-anchor/
   ├── index.js
   ├── anchor.html
   ├── style.css
   ├── manifest.json
   └── README.md（这个是使用说明，虽然也没什么好看的）
   ```
2. 重启 SillyTavern 或按 F5 刷新页面。

🚀 使用方法

打开设置面板

有两种方式可以打开一致性锚的设置面板：

1.🪄 魔法棒（扩展程序）：点击聊天界面底部的魔法棒图标，在弹出的菜单中点击 "一致性锚设置"
2.⚓ 悬浮按钮：点击屏幕右下角的锚形 (⚓) 悬浮按钮（可自由拖拽位置）

配置说明
（均可自行修改）
启用插件：开启/关闭所有功能。默认开启
锚点注入频率：每隔 N 轮对话注入一次设定锚点（防止 AI 遗忘角色设定）。默认3轮。
记忆池最大 Token：从高优先级世界书中提取的记忆最大长度。默认200
重复检测缓存数：保留最近 N 条 AI 回复用于检测重复。默认3条。
重复判定阈值：相似度超过此值判定为复读（0~1，数值越大越宽松）。默认为0.70。
自动修正轻微格式偏差：自动修正引号、星号等格式问题。默认开启。
严格格式检查：启用后会在控制台输出更详细的格式警告。默认开启。
启用思维链拦截：检测并警告 AI 是否泄露思考过程。默认开启。
调试模式：在浏览器控制台输出详细的运行日志。默认关闭。
开启自由拖拽悬浮图标：显示/隐藏右下角的锚形悬浮按钮。默认开启显示。

💡 提示：所有设置修改后会自动保存，无需手动点击保存按钮。（这个保存按钮单纯做出来图安心，其实一直都是你改了啥都自动保存。）

❓ 可能遇到的问题

#Q1：右下角的悬浮按钮不见了怎么办？
· 检查设置面板中 "开启自由拖拽悬浮图标" 是否勾选。
· 如果关闭了，可以通过底部"魔法棒"菜单入口打开面板重新开启。
· 如果仍然不显示，请尝试刷新页面。

#Q2：为什么我拖拽悬浮按钮后，下次打开位置没有保存？
· 插件会自动保存位置。如果发现位置没有记忆，请检查浏览器是否允许网站存储本地数据（LocalStorage）。
· 也可以检查浏览器控制台是否有相关错误日志。

#Q3：插件检测到"思维链泄露"但实际是 AI 正常的思考性对话？
· 思维链检测基于关键词匹配（如"思考：""推理："等）。如果 AI 在角色扮演中自然地使用这些词（如角色说"让我思考一下"），可能会被误报。
· 如果误报频繁，可以在设置中关闭 "启用思维链拦截"，或调整为更宽松的检测条件。

#Q4：格式修正没有生效？
· 格式修正依赖正确的"角色规范"解析。如果角色描述中缺少明确的格式指引（如未使用 *动作* 或 "对话" 模式），插件可能无法准确修正。
· 如果环境不支持自动更新消息（如某些旧版 ST），控制台会输出格式修正建议，你可以手动编辑消息。

#Q5：插件会影响其他扩展或性能吗？
· 插件采用安全注入机制，不会覆盖其他扩展的内容。
· 所有检测和修正逻辑均在消息接收后执行，不会阻塞聊天生成流程，对性能影响极小。

#Q6：支持哪些版本的 SillyTavern？
· 最低支持 SillyTavern 1.13.0（主版本号 ≥ 1.13.0 均可）。
· 旧版本可能部分功能不兼容，建议升级版本。
· 因为朋友还在使用1.13版本的st所以想办法做了降级（拳击拷打！！），但其他版本是否兼容不太确定，因为实验小白鼠（bushi）太少了。
· 总共做了两版，一个是自己用的适配1.18，一个是给朋友的适配1.13。大家自己选一下安装哪一个吧。
